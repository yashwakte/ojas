# ओजस OJAS — शुद्ध आहार...शुद्ध विचार

> Premium quality flours & healthy food products — crafted with tradition, delivered with love.

## Tech Stack

| Layer     | Technology                          |
|-----------|-------------------------------------|
| Frontend  | Angular 21, Angular Material, SCSS  |
| Backend   | .NET 10 Web API                     |
| Database  | MongoDB Atlas                       |
| Auth      | JWT Bearer Tokens, BCrypt           |

## Features

- Modern responsive UI with glassmorphism & animations
- 13 product catalog with category filtering
- JWT authentication (Register / Login)
- RESTful API with MongoDB Atlas backend
- Mobile-first responsive design

## Getting Started

### Backend
```bash
cd ojas-api
dotnet run --launch-profile https
```
API runs at `https://localhost:7126`

### Frontend
```bash
cd ojas-frontend
npm install
ng serve --open
```
App runs at `http://localhost:4200`
